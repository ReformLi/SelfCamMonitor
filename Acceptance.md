# 验收清单（对照需求文档）

### ✅ 1. 实时视频流

- 浏览器访问 `http://手机IP:8080/video`，看到低延迟实时画面。
- 熄屏、后台仍运行。

### ✅ 2. 运动检测及报警

- 挥手触发“检测到运动”日志。
- HTTP 报警功能已实现（地址可配置）。
- 灵敏度调节：已留接口，可在 `MotionDetector.setSensitivity()` 使用。

### ✅ 3. 运动触发短视频

- 每次运动开始录制 10 秒 MP4，存放于 `Android/data/.../Recordings/`。
- 录制与推流互不干扰，不卡顿。

### ✅ 4. 控制界面

- 显示 IP、运行状态。
- 开关：MJPEG 推流 / 运动检测。
- 显示存储使用情况。

### ✅ 5. 后台保活

- 前台服务，通知栏常驻。
- CPU WakeLock 持有。
- 电池优化请求已实现。

## 最后一步：生成 APK

1. **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**。
2. 完成后，APK 位于 `app/build/outputs/apk/debug/`。
3. 安装到旧手机，即可作为专用监控摄像头使用。
