# 旧手机摄像头 App 操作手册

## 1. 应用概览

本应用可将您的旧 Android 手机变成一台局域网监控摄像头。只需在一部手机上安装并运行本应用，同一 Wi‑Fi 网络下的其他设备（手机、电脑、平板）即可通过浏览器观看实时画面，同时应用支持运动检测、自动录像、报警推送等功能。

### 主要特性

- 实时视频流（MJPEG）观看，支持多设备同时访问
- 运动检测，灵敏度可调
- 运动触发短视频录制（可配置时长）
- 连续录像并自动分段保存（可配置分段时长）
- 录像文件按日期自动分文件夹管理
- 录像列表浏览、播放、搜索、批量删除/导出
- 后台保活（前台服务 + 唤醒锁）
- 支持开机自启（需部分手机手动授权）
- 可选 HTTP 基本认证，保护视频流

## 2. 安装与权限

- 安装 APK 后，打开应用。
- 首次点击 **“启动监控”** 会依次请求：
  - 相机权限（必须）
  - 录音权限（可选，用于录制有声视频）
  - 忽略电池优化（建议允许，否则后台可能被系统杀死）
- 若未授予必要权限，服务无法启动。

## 3. 主界面（MainActivity）

主界面是启动后的默认页面，用于控制监控服务、查看状态和跳转其他功能。

### 3.1 服务状态与 IP 地址

- **IP 地址**：显示手机当前 Wi‑Fi IP（如 `192.168.1.5`）。浏览器访问视频流使用该地址。
- **服务状态**：绿色“服务运行中”表示正在监控；红色“服务已停止”表示未启动。
- **录像存储**：显示所有录像文件占用的总空间（MB），每次回到主界面自动刷新。

### 3.2 功能控制

- **MJPEG 推流开关**：开启后，浏览器可访问 `http://手机IP:8080/video` 观看实时画面（需用户名密码则输入）。关闭则推流停止，但相机仍可用于检测和录像。
- **录像模式选择**：通过两个互斥开关实现三档模式。
  - **连续录像**：开启后持续录制，按“设置”中的分段时长自动分割文件。
  - **运动触发录像**：开启后仅在检测到运动时录制短视频，时长可在设置中调整。
  - 两个开关同时关闭 = **仅预览**（不录像、不运动检测）。
- **启动监控**：检查权限后启动后台服务，通知栏出现“监控运行中”。
- **停止监控**：停止服务，释放摄像头。
- **查看录像**：进入录像管理页面。
- **设置**（右上角齿轮）：进入详细配置页面。

### 3.3 录像管理入口

点击 **“查看录像”** 进入 `RecordingsActivity`，按日期文件夹管理所有录像。

## 4. 设置界面（SettingsActivity）

设置界面包含四个区块：

### 4.1 视频设置

- **推流分辨率**：从摄像头支持的尺寸中选择（如 640x480、1280x720）。**需重启监控生效**。
- **推流帧率**：滑块 1~16 fps。实际受 CameraX 限制，上限约 16 fps。**需重启监控生效**。

### 4.2 运动检测设置

- **检测灵敏度**：0（最敏感）～ 100（最迟钝）。数值越低，微小运动也会触发。
- **报警静默期**：两次报警的最小间隔（秒），避免频繁推送。
- **报警接收 URL**：运动触发时，应用会向该地址发送 HTTP GET 请求，携带 `timestamp` 参数。例如 `http://192.168.1.100:3000/motion?timestamp=...`。

### 4.3 录像设置

- **录像模式**：与主界面开关同步，可在此切换。
- **运动录像时长**：可选 10秒 / 30秒 / 1分钟 / 3分钟 / 5分钟。
- **连续录像分段时长**：可选 1分钟 / 5分钟 / 10分钟 / 30分钟 / 1小时。修改后不影响当前正在录制的片段，下次分段生效。

### 4.4 系统设置

- **开机自启**：开启后，手机开机时自动启动监控服务（部分厂商需手动授予自启动权限）。
- **监控时间限制**：设置每日起止时间（如 08:00 – 20:00），非监控时段相机不分析也不录像。

### 4.5 安全设置

- **HTTP 认证用户名/密码**：设置后，访问视频流需输入用户名和密码（浏览器弹出验证框）。留空则不认证。

所有设置（除分辨率、帧率外）修改后立即生效，无需重启服务。

## 5. 录像管理 – 文件夹列表（RecordingsActivity）

点击主界面“查看录像”后进入，按日期文件夹（格式 `YYYY-MM-DD`）展示所有录像。



### 5.1 普通模式

- 顶部显示“共 N 个文件夹”。
- 点击任意文件夹进入该日期的视频列表。
- 长按文件夹进入多选模式。

### 5.2 多选模式

- 长按任意文件夹进入，顶部变为：
  - 左侧“全选”按钮
  - 中央“已选 X 项”
  - 右侧“取消”按钮
- 每个文件夹左侧出现复选框，可勾选多个。
- 底部显示“删除选中”和“导出选中”按钮。
- **删除**：弹出确认框，删除选中文件夹及其内所有视频（不可恢复）。
- **导出**：选择外部目录（通过系统文件选择器），递归复制选中文件夹及内部视频文件到目标位置。支持 Android 10+ 分区存储。
- 按返回键退出多选模式。

## 6. 视频列表页面（VideoListActivity）

点击任意日期文件夹后进入，展示该文件夹内所有 MP4 视频文件。

### 6.1 普通模式

- 顶部：左侧返回按钮，中央文件夹名，右侧视频数量。
- 每个视频项显示：文件名、大小（KB）、录制日期时间、视频时长（mm:ss 或 HH:mm:ss）。
- 短按视频调用系统播放器播放。
- 长按视频进入多选模式。

### 6.2 搜索功能

- 搜索栏始终显示在列表上方。
- 输入关键词，点击“搜索”按钮或键盘搜索键，列表实时过滤（文件名包含关键词）。
- 输入框有文字时，右侧出现“✕”清除按钮：
  - 点击“✕”清空输入框，并立即恢复显示全部视频（焦点保留）。
  - 输入框为空时点击“✕”则隐藏键盘并失去焦点。
- 搜索结果不影响元数据缓存。

### 6.3 多选模式

- 长按任意视频进入多选模式，顶部变为：
  - 左侧“全选”按钮
  - 中央“已选 X 项”
  - 右侧“取消”按钮
  - 同时隐藏返回按钮和文件计数。
- 每个视频左侧出现复选框，点击可切换选中。
- 底部显示“删除选中”和“导出选中”按钮。
- **删除**：确认后批量删除选中视频（不可恢复）。
- **导出**：选择外部文件夹，复制选中视频文件到目标位置。
- 按返回键退出多选模式。

### 6.4 播放视频

短按视频后，系统会弹出可选播放器（如系统图库、VLC等），点击即可观看。录制的视频包含音频（如果授予了录音权限）。

## 7. 实时视频观看（浏览器）

在手机启动监控服务后，同一 Wi‑Fi 下的任意设备（电脑、另一手机）打开浏览器，输入：

```
http://手机IP地址:8080/video
```

例如 `http://192.168.1.5:8080/video`。

- 如果设置了 HTTP 认证，浏览器会弹出窗口要求输入用户名和密码。
- 页面显示实时 MJPEG 流，刷新率由推流帧率决定。
- 支持多设备同时观看，互不影响。

## 8. 常见问题与提示

### 8.1 服务无法启动

- 检查是否授予相机权限。
- 部分手机需要手动将应用加入“受保护应用”或“允许后台活动”列表。
- 尝试重启手机后再启动服务。

### 8.2 录像文件找不到

- 录像保存在内部存储：`Android/data/com.hpu.selfcammonitor/files/Recordings/`。
- 可通过应用内“查看录像”浏览，也可用文件管理器访问（Android 11+ 可能受限）。

### 8.3 运动检测不灵敏

- 进入设置调低灵敏度（如 20）。
- 确保摄像头对着有变化区域（不要对准完全静止的墙面）。

### 8.4 报警未收到

- 检查设置的报警 URL 是否可访问（如局域网内另一台电脑的 web 服务）。
- 确认手机与目标服务器在同一网络，且没有防火墙拦截。

### 8.5 连续录像分段时长修改后不立即生效

- 当前正在录制的分段会继续使用旧时长，结束后新分段才应用新时长。这是设计行为，避免文件损坏。

### 8.6 导出文件夹时提示失败

- 目标目录必须是可写的（如 Download、Documents 等用户可访问的位置）。
- 不要选择应用自身的数据目录（可能无权限）。
